Frequently Asked Questions

Find answers to some of the most common questions about CryptoTrack.

What is CryptoTrack?

CryptoTrack is a cryptocurrency portfolio tracker that allows users to monitor their holdings in real time using price data from CoinGecko.

Is CryptoTrack free to use?

Yes, CryptoTrack is completely free. It is built as an educational project and hobby tool for crypto enthusiasts.

How does CryptoTrack get its price data?

We use the CoinGecko API to fetch real-time prices for various cryptocurrencies in USD.

Can I track multiple coins?

Yes! You can add multiple coins with their respective amounts, and the tool will calculate the total value of your holdings.
